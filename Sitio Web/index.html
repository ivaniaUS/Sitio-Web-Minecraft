MINECRAFT

Minecraft es un videojuego de construcción de tipo «mundo abierto», creado originalmente por el
sueco Markus Persson, que posteriormente creo Mojang Studios (actualmente parte de Microsoft).
Está programado en el lenguaje de programación Java para la versión Java Edition y posteriormente
desarrollado en C++ para la versión de Bedrock Edition. Fue lanzado el 17 de mayo de 2009, y des-
pués de numerosos cambios, su primera versión estable fue publicada el 18 de noviembre de 2011.
